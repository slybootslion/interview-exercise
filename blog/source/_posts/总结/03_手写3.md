### 01 模拟Object.create

Object.create()方法创建一个新对象，使用现有的对象来提供新创建的对象的__proto__。
说人话就是，新对象的原型链指向现有对象（继承）。

```js
function createObj(obj) {
  function F() {}
  F.prototype = obj
  return new F()
}
```
